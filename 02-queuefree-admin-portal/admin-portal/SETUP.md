# Admin Portal Setup

## Step 1 - Install packages
```
npm install
```

## Step 2 - Start the portal
```
npm start
```

Opens at: http://localhost:3000

NOTE: The backend must be running first on port 5000.
The proxy in package.json forwards /api calls to localhost:5000 automatically.
